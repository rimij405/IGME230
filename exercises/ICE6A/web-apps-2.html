Web Application Exercise 2

Please take a look at the developer's console for this page.

Tasks

This page accomplishes the following:

  1. Use a method of the Array object to append another color to the end of the colors array.
  2. Print out the last element in the colors array.
  3. Loop through the colors array using a for loop and print out each value to the console.
  4. Add a new property named school to the person object and give it a value of "RIT". Then print this value to the console.
  5. Print out the number of seconds that have passed since 1970 - use the Date object.
  6. Print out the value of pi - use the Math object.
  7. Print out the absolute value of -999 - use the Math object.
  8. Print out an "all caps" version of the string "Hello" - use a method of the String object.
