Tasks

This page accomplishes the following:

  1. Add your favorite colors and foods to the arrays.
  2. Write code that loops through the colors array, generates list items, and appends them to the appropriate list element. For this array use a classic for loop.
  3. Write code that loops through the foods array, generates list items, and appends them to the appropriate list element. For this array use an ES6 for...of loop.
  4. Be sure that your code uses document.createElement() to create each element.
  5. Optional: add your favorite web sites to the links object literal, and then loop through the object, pulling out both the key and the value, generate functioning links, and add them to the last ‹ul› on the page. This is worth an extra 5 points on the HW assignment. The JavaScript for...in loop will probably get the job done.
