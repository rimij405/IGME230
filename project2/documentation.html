Project Documentation

How it's made.

This document contains a reference to how the Minesweeper game was made.

Instructions for playing the game can be found on the project page itself.

I had originally wanted to make a more interactive experience involving sound, but, I decided to focus on getting the core functionality working.

I am very pleased with the outcome of this project.

Requirements

Does it fulfill them?
  • The minesweeper game is functional; it has sound so turn up that volume!
  • Code is organized neatly and placed in a separate file for ease of access.
  • Both HTML and JavaScript are commented; setting debugmode to true in the source files will also show verbose comments.
  • This CSS is pretty fly. So is the JavaScript's use of data(set) attributes.
  • It works and was tested in Chrome; no guarantees for other browsers. (Although they are just as loved!)
  • I worked on this alone.
